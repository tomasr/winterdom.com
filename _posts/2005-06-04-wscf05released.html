My good friend Christian Weyer and the fellows at Thinktecture have released version 0.5 of WSCF.

This is a very exciting release I've been expecting for a while now (I was one of the lucky testers of beta versions of it), and I promise you'll be excited about it as well once you check out all the cool new features. My current favourites are: Separate File code generation, the new doc page generated for the services (similar to the standard ASMX one), multiple schema file support and the use of interfaces for code generation.
