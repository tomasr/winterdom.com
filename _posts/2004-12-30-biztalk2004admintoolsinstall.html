I'm testing a few BizTalk Server 2004 installation variants lately, and ran into a snag. What I was trying to do was install BizTalk Server on one machine, and then setup a second machine with only the administration tools.

According to the Installation Guide, on the section "Performing a Custom Installation", if you only install the Admininstration and Monitoring Tools, SQLXML 3.0 is not a needed prerequisite.

However, no matter what I do, the BizTalk Installer keeps asking for SQLXML to be installed on the machine where the administration tools will go. Is the documentation wrong about this point, or am I doing something incorrectly?
