While searching the ECMA site for the ECMAScript spec the other day, I ran into a really unexpected specification: "ECMA-234: Application Programming Interface for Windows (APIW)"
This one really caught me by surprise! OK, did anybody else expect to find the Win3.1 API standarized?
Please note the date on this document (dec. 1995) is after Win95 was already out...
