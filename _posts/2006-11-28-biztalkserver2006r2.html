Richard Seroter has posted a quick guide on the BizTalk 2006 R2 Beta 1 installation, showing a few of the new features included. Pretty cool stuff. Meanwhile, Jesus Rodriguez posted a cool overview on how to use the WCF adapter in R2 to invoke a WCF service using the NetTcpBinding.

I'm very excited about the new features announced in R2, particularly about the WCF adapter and the WCF adapter framework, and I'm hoping to get my hands dirty with it in the future.

BTW, I've been giving these past few weeks short BizTalk Clinics with Microsoft costumers as well as longer trainings with developers, and WCF integration with BizTalk has been one of the topics I was most often asked about, so there's definitely some healthy interest on these topics!
