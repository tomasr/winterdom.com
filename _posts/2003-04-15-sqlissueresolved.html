I just managed to resolve the issue I was having with SQL Server 2000 on Windows Server 2003. Thanks go to Steve Kass (a fellow MVP), who refered me to KB Article 309398.

The problem was that I had installed on my machine Certificate Server before installing SQL. Once I uninstalled it, I was able to install SQL without a hitch, after which I just reinstalled CertSrvr.
