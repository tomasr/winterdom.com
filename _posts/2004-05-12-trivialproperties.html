One of the features I'm liking more in C++/CLI is Trivial Properties:

#using 

using namespace System;

ref class X

{

public:

property int Value;

};

int main()

{

X^ x = gcnew X();

x->Value = 4;

Console::WriteLine("Value is {0}", x->get_Value());

}

Pure syntactic-sugar, but saves a lot of typing...
