Microsoft has been kind enough to allow me to provide a download site for the error message databases for Microsoft BizTalk Server 2004, to use with the excellent MessageLookup tool created by fellow MVP Raffaele Rialdi. The databases are available in four languages: French, Italian, Spanish and German.

I'd like to thank Jorge Aguilar Sanchez from Microsoft for making the necessary arrangements within Microsoft to allow me to host these files. Enjoy!
