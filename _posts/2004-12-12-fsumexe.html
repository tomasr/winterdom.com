Yesterday, I was looking for an utility to calculate a CRC value for a set of files I needed, and ran across FSUM; a very nice [free] command line utility that can calculate checksum values for a file (or a set of them) using any of 13 different algorithms: MD2, MD4, MD5, SHA1, SHA256, SHA384, SHA512, RIPEMD-160, TIGER, ADLER32, CRC32 and the EDonkey algorithm.

All in all, a very useful utility to keep handy!
