Earlier this week I upgraded to Microsoft Virtual PC 2004 from VPC 5.2. Looks pretty nice, and seems to be working fine. In fact, networking finally works right for me, so that by itself is a huge improvement!

A few months ago, I reported about a conflict between VPC 5.2 and Outlook 2003. Well, with 2004, it seemed to be gone. Virtual machines start up with no problems when outlook is open.

Unfortunately, the problem seems to have reversed itself. Now, instead of having VPC hang while the VM is starting up, it hangs when the Virtual Machine is shutting down!!! GRRRR!!!!
